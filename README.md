# Mahir's Portfolio

A high-performance, responsive personal portfolio built with the Next.js App Router, TypeScript, and Tailwind CSS.

**Official Website**: https://mahirashef12.vercel.app/

---

## 🚀 Tech Stack

- [Next.js](https://nextjs.org/) – React framework for production.
- [TypeScript](https://www.typescriptlang.org/) – Static type checking for robust code.
- [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework for rapid UI development.
- [Vercel](https://vercel.com/) – Deployment and hosting platform.
- [Spline](https://spline.design/) - An interactive WebGL engine powering models and animations.

---

## 📂 Project Structure

This project follows modern Next.js conventions for scalability and maintainability:

```text
.
├── app/          # Routes, layouts, and page-specific logic
├── components/   # Reusable UI components
├── hooks/        # Shared custom React hooks
├── public/       # Static assets (images, icons, resume, etc.)
├── next.config.ts
├── tsconfig.json
└── package.json
```

## 🛠️ Getting Started

Set up the project locally in your System.

### 1) Clone the repository

```bash
git clone https://github.com/Mahir-o4/Portfolio.git
```
```bash
cd Portfolio
```

### 2) Install dependencies

```bash
npm install
```

### 3) Set up environment variables

Create a `.env` file in the project root.

```bash
cp .env.example .env
```

### 4) Run the development server

```bash
npm run dev
```

### Open [http://localhost:3000](http://localhost:3000).
